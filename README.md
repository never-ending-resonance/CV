# Haosong Xu Resume (Static HTML)

Single-page resume built with plain HTML/CSS, matching the provided design.

## Files
- `index.html` — main page
- `logo.png` — top-left logo image
- `formula.png` — formula image in the hero card

## Preview locally
Option 1: open directly  
- Double-click `index.html`

Option 2: run a simple server (from this folder)
```bash
python -m http.server 8000
```
Then open `http://localhost:8000` in your browser.

## Deploy to GitHub Pages
1) Create a repo and push these files.  
2) In the repo settings, enable GitHub Pages with the `main` branch root.  
3) The site will be available at `https://<your-user>.github.io/<repo>/`.
